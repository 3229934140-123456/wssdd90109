## 1. 产品概述

面向企业公关团队的 Web 口碑观察站，用于集中监控论坛、贴吧中关于品牌、产品和高管的讨论变化，帮助公关团队快速识别新增讨论、热度上升和负面集中话题，并提供处置记录与日报生成能力，实现从"发现"到"处置"到"汇报"的闭环管理。

- 目标用户：企业公关团队成员、公关负责人
- 核心价值：将分散的论坛讨论聚合可视化，缩短从发现舆情到处置响应的时间，日报自动汇总降低早会准备成本

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 公关专员 | 管理员分配账号 | 录入关键词、浏览帖子、标记处置、查看日报 |
| 公关负责人 | 管理员分配账号 | 全部专员权限 + 筛选策略配置、日报审批导出 |

### 2.2 功能模块

1. **首页（关键词看板）**：关键词录入、今日变化概览（新增讨论/热度上升/负面集中）、快速入口
2. **帖子列表页**：多维筛选、帖子分层展示、原帖摘要、处置标记与结论填写
3. **日报生成页**：高频槽点汇总、典型原帖链接、风险等级评估、一键导出

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 首页 | 关键词管理面板 | 录入品牌名、产品别名、常见错别字、竞品词，支持分组与标签颜色 |
| 首页 | 今日变化看板 | 按"新增讨论、热度上升、负面集中"三个维度展示今日变化卡片，点击进入对应筛选列表 |

| 帖子列表页 | 筛选栏 | 按吧名、论坛板块、情绪倾向（正面/中性/负面）、回复速度筛选 |
| 帖子列表页 | 内容分层标签 | 投诉、求助、吐槽、推荐四类分层标签，支持多选 |

| 日报生成页 | 导出区 | 一键导出日报（PDF/打印），预览日报全文 |

## 3. 核心流程

用户在首页录入关键词 → 系统自动按关键词匹配讨论数据 → 首页看板展示今日变化概览 → 用户点击变化卡片进入帖子列表 → 通过筛选和分层定位目标帖子 → 点击帖子查看原帖摘要 → 标记处置状态并填写结论 → 公关负责人在日报页查看汇总 → 确认风险等级后导出日报

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：深炭灰（#1a1a2e）+ 琥珀警示色（#f59e0b）+ 翠绿安全色（#10b981），搭配暗色背景营造监控中心氛围
- 按钮风格：圆角微凸，轻微阴影，状态色区分（观察-蓝、客服跟进-琥珀、法务关注-红）
- 字体：标题使用 Source Han Sans / Noto Sans SC Bold，正文使用 Noto Sans SC Regular，数据数字使用 JetBrains Mono
- 布局：左侧固定导航栏 + 右侧内容区，卡片式内容组织，顶部统计条
- 图标风格：线性图标（lucide-react），统一 18px 尺寸

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 首页 | 关键词管理面板 | 侧滑抽屉式面板，标签分组排列，颜色编码，支持拖拽排序 |
| 首页 | 今日变化看板 | 三列卡片布局，每张卡片大号数字+趋势小图标，点击整张卡片跳转 |
| 首页 | 实时动态流 | 紧凑卡片流，左对齐时间轴，每条含来源图标+标题+情绪色条 |
| 帖子列表页 | 筛选栏 | 顶部固定横条，下拉多选+滑块+标签切换组合 |
| 帖子列表页 | 帖子卡片 | 左侧情绪色条，右侧信息区，底部标签行，hover 展开微动效 |
| 帖子列表页 | 原帖摘要弹窗 | 居中模态，顶部原帖标题+来源，中部摘要区，底部处置操作区 |
| 日报生成页 | 高频槽点区 | 词云可视化 + 右侧 TOP10 列表，字号映射频次 |
| 日报生成页 | 典型原帖区 | 编号列表，标题可点击跳转，行尾风险色点 |
| 日报生成页 | 风险等级面板 | 大号仪表盘式指示器，色彩渐变，居中突出 |
| 日报生成页 | 导出区 | 底部固定操作栏，预览+导出双按钮 |

### 4.3 响应式

- 桌面优先设计，主要针对 1440px+ 宽度优化
- 平板（768px-1024px）：导航栏收缩为图标模式，卡片从三列变两列
- 移动端：底部标签导航，单列卡片，筛选栏折叠
